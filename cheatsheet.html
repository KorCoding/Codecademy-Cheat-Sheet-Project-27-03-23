Combining CSS SELECTOR

Name: Descendant Combinator
Description: Matches any B element that is a descendant (inside) of an A element.

Pattern Code Example Match Not Match
A B h2 p { color: blue; } 
    <h2>
        <p> Blue </p>
        <div>
            <p> Still Blue </p>
        </div>
    </h2>
                        
    <h2> </h2>
    <p> Not Blue </p>

Name: Child Combinator
Description: Matches any B element that is a child of A ( Not Grandchilds).

Pattern Code Example Match Not Match
A > B h2 > p { color: blue; } 
    <h2>
        <p> Blue </p>
    </h2>
                        
    <h2>
        <div>
            <p> Not Blue </p>
        </div>
    </h2>

Name: Next sibling combinator
Description: Matches if A and B share the same parent and A immediately precedes B.

Pattern Code Example Match Not Match
A + B h2 + p { color: blue; } 
    <h2> </h2>
    <p> Blue </p>
                        
    <h2> </h2>
    <h3> </h3>
    <p> Not Blue </p>

Name: Compounding multiple class or ID selectors
Description: Matches if A and B or more in the same tag. (Can be id,class,type)

Pattern Code Example Match Not Match
.class1.class2(.classN) .a.b { color: blue; } 
    <div class="a b">
        <p> Blue </p>
    </div>
                        
    <div class="a">
        <p class="a" > Not Blue </p>
    </div>

Name: Following Sibling Combinator
Description: Matches when an B element is preceded by an A element (not necessarily immediately, as in case of A > B).

Pattern Code Example Match Not Match
A ~ B h2 ~ p { color: blue; } 
    <h2> </h2>
    <h3> </h3>
    <p> Blue </p>
                        
    <h2>
        <p> Not Blue </p>
   </h2>
